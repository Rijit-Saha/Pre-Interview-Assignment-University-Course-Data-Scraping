# Advanced University & Course Scraper (Selenium Version)

## Features
- Uses Selenium (Headless Chrome)
- Automated scraping
- Relational integrity maintained
- Duplicate handling
- Clean 2-sheet Excel export

## Setup Instructions

1. Install Google Chrome
2. Install ChromeDriver (matching version)
3. Install dependencies:
   pip install -r requirements.txt

4. Run:
   python scraper.py

## Output
University_Course_Data.xlsx
